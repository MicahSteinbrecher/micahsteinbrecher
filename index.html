Inventory Tracker

This is a prototype I created to help diamond vendors maintain data on their inventories. Users create an inventory and input stone data, which gets evaluated against the Rapaport Price Index. The Tracker automates the process of rechecking inventory value with market fluctuations.

Chorale Analyzer

A free, open source tool I collaborated on for performing error analysis on four part chorales. It is intended to help students and teachers analyze the chorale in an academic setting by automating the formerly tedious chorale checking process. Chorale Analyzer uses the open source Sebastian library to perform analysis, and the open source music21 library to read and parse musical data.
