Inventory Tracker

Inventory Tracker is a prototype that helps diamond vendors keep track of their inventories. Think an internet accessible excel spreadsheet. Users create an inventory and input stones, which Tracker evaluates against the rapport price index (they determine the market value of diamonds). When vendors want to value their liquidity, they run into a problem with fluctuations in the diamond market. A store owner who did inventory during new years could only estimate what his value is today. The Tracker updates value as the market changes to avoid this problem.

Chorale Analyzer

Chorale Analyzer is a free, open source tool for performing error analysis on four part chorales. It is intended to help students and teachers analyzing the chorale in an academic setting by automating the tedious task of checking chorales for errors. Chorale Analyzer uses the open source Sebastian library to perform analysis, and the open source music21 library to read and parse musical data.
